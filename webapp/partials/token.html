Plik 令牌允许您在没有源 IP 限制的情况下上传文件。

  • 令牌只能从有效的源 IP 生成。
  • 您可以通过点击记住按钮将令牌保存到浏览器的本地存储中。
  • 如果您使用命令行客户端， 可以通过在 ~/.plikrc 文件中添加 Token = "xxxx" 行来使用令牌。
  • 您可以使用浏览按钮列出令牌拥有的所有上传。
